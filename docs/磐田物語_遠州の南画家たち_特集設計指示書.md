## 1. 企画概要

本設計書は、「磐田物語」の新しい文化・美術特集として「遠州の南画家（文人画家）たち」を立ち上げるためのHTML実装およびデザイン設計について定義したものである。

磐田（遠州地域）は、江戸時代後期から明治期にかけて、多くの文人画（南画）の巨星や、それを支えた豪商・文化人のネットワークが存在した土地である。特に旧福田町（ふくで）出身で渡辺崋山門下の俊英となった**福田半香（ふくだはんこう）**や、幕末の激動期に志士たちと交わった**平井顕斎（ひらいけんさい）**など、地域に根ざしながら日本の美術史に足跡を残した人々がいる。

本特集では、彼らの絵画作品や詩文を単なる骨董としてではなく、**「中遠の豊かな経済力（掛塚・見付の豪商パトロン）」「江戸・京都と繋がった知識人の交流網」「現地の風景」**という独自の地域史視点から再構成し、ウェブ上で美しく表現する。

## 3. デザインシステム（色彩・タイポグラフィ）

南画・文人画の持つ「水墨」「落款（印影）」「余白の美」をウェブデザインに昇華させ、通常の地域別ページよりもさらに静謐で知的な印象を与えるビジュアルスタイルを設計する。

### 3.1 カラーパレット (南画専用CSS変数)

```css
:root {
  /* 基本背景・テキスト */
  --nanga-bg-washi: #f8f7f4;       /* 和紙の白（薄い生成り、背景色） */
  --nanga-text-ink: #1a1a1a;       /* 生墨（非常に深いグレー黒、本文） */
  --nanga-text-muted: #5a6268;     /* 薄墨（補助テキスト、メタ情報） */

  /* アクセントカラー */
  --nanga-accent-seal: #8d3c26;    /* 落款朱（落款・印影の赤、重要アクセント） */
  --nanga-border-ink: #d2cbb8;     /* 古紙金泥（表装・掛軸のヘリをイメージした境界線） */
  --nanga-bg-panel: #efece5;       /* 泥絵具（コラムや図版解説の背景） */
}
```

### 3.2 タイポグラフィと余白

- **見出し (`h1`, `h2`, `h3`)**:
  見出しには、筆の勢いと古典的な美しさを併せ持つ **Shippori Mincho (しっぽり明朝)** を太字（`font-weight: 800`）で適用する。
- **本文**:
  墨と和紙の対比を美しく見せるため、文字と文字の間隔（`letter-spacing`）を `0.05em` と少し広めに設定し、行間（`line-height`）は `1.9` 以上を確保する。
- **余白**:
  「余白の美」を意識し、セクション間の余白（`margin-bottom`）は既存ページより 1.5倍 広く設計する。

## 4. 特集専用コンポーネント

記事の魅力を高めるため、以下のHTML/CSS構造を持つ特集専用コンポーネントを新規に定義する。

### 4.1 落款マーク (`.seal-stamp`)
各記事の導入部や末尾、または署名欄に表示する赤正方形の落款（スタンプ）風の装飾。
```html
<div class="seal-stamp">半香</div>
```
```css
.seal-stamp {
  display: inline-block;
  border: 2px solid var(--nanga-accent-seal);
  color: var(--nanga-accent-seal);
  font-family: "Shippori Mincho", serif;
  font-weight: bold;
  padding: 4px 6px;
  line-height: 1;
  font-size: 0.9rem;
  letter-spacing: 0.1em;
  text-transform: uppercase;
  background-color: transparent;
  transform: rotate(-2deg); /* わずかに手押し風に傾ける */
}
```

### 4.2 掛軸風図版ギャラリー (`.nanga-gallery`)
絵画作品を美しく展示するための枠線デザイン。
```html
<figure class="nanga-gallery-item">
  <div class="kakejiku-frame">
    <img src="/images/nanga-hanko-sansui.jpg" alt="福田半香 山水図">
  </div>
  <figcaption class="nanga-caption">
    <span class="artwork-title">山水図</span>（個人蔵・部分）<br>
    <span class="artwork-meta">福田半香 筆 ｜ 天保年間 ｜ 紙本墨画淡彩</span>
    <p class="artwork-desc">半香が得意とした、細密な筆致と澄んだ淡彩が特徴の山水画。崋山から学んだ西洋画風の陰影表現が山石の立体感に活かされている。</p>
  </figcaption>
</figure>
```
```css
.kakejiku-frame {
  border: 8px solid var(--nanga-border-ink);
  background-color: #ffffff;
  padding: 12px;
  box-shadow: 0 4px 10px rgba(0,0,0,0.05);
  margin-bottom: 12px;
}
.nanga-caption {
  font-size: 0.9rem;
  color: var(--nanga-text-muted);
  border-left: 2px solid var(--nanga-accent-seal);
  padding-left: 10px;
}
.artwork-title {
  font-weight: bold;
  color: var(--nanga-text-ink);
}
```

## 5. 品質および検証基準

1. **文章クオリティ**:
   - 磐田物語の基本ルールに基づき、文章はすべて「である調」。
   - 最低文字数は各詳細記事 **2,500文字以上** とし、単なる作品リストではなく、人物の「生きた文脈」を描ききる。
2. **画像アクセシビリティ**:
   - 美術品画像には必ず適切な `alt` 属性を記述し、作品名と作者名を明記する。
   - 所蔵先や図版の権利（パブリックドメイン・許可済等の出典）を `.sourcebox` 内に漏れなく記載する。
3. **技術SEO**:
   - 記事タイトルは「[記事名] ─ 遠州の南画家たち ｜ 磐田物語」のように一貫性を持たせる。
   - `sitemap.xml` への新規5URLの登録と、`/data/pages.json` へのレコード追加を行う。
